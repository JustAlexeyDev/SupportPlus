## Технологический стек

### Frontend (Клиентская часть)

**Основные технологии:**
- **React** 19.2.0 — библиотека для построения пользовательского интерфейса
- **React Router** 7.9.6 — маршрутизация и навигация в SPA
- **React Scripts** 5.0.1 — инструменты сборки (Create React App)

**HTTP и API:**
- **Axios** 1.13.2 — HTTP-клиент для взаимодействия с backend API

**Аутентификация и безопасность:**
- **js-cookie** 3.0.5 — работа с cookies для хранения токенов
- **crypto-js** 4.2.0 — криптографические функции

**UI компоненты:**
- **lucide-react** 0.553.0 — библиотека иконок

**Языки программирования:**
- **JavaScript (ES6+)** — основной язык разработки
- **JSX** — синтаксис для компонентов React

### Backend (Серверная часть)

**Основной фреймворк:**
- **NestJS** 10.3.0 — прогрессивный Node.js фреймворк для построения эффективных и масштабируемых серверных приложений
- **Express** (через @nestjs/platform-express) — веб-фреймворк для Node.js

**ORM и база данных:**
- **TypeORM** 0.3.17 — Object-Relational Mapping для работы с БД
- **SQLite3** 5.1.6 — легковесная реляционная база данных

**Аутентификация и авторизация:**
- **Passport.js** 0.7.0 — middleware для аутентификации
- **@nestjs/passport** 10.0.2 — интеграция Passport с NestJS
- **passport-jwt** 4.0.1 — стратегия JWT для Passport
- **passport-google-oauth20** 2.0.0 — стратегия Google OAuth 2.0
- **@nestjs/jwt** 10.2.0 — модуль JWT для NestJS
- **bcrypt** 5.1.1 — хеширование паролей

**Валидация и трансформация данных:**
- **class-validator** 0.14.0 — декораторы для валидации DTO
- **class-transformer** 0.5.1 — трансформация объектов

**Конфигурация:**
- **@nestjs/config** 3.1.1 — модуль для работы с переменными окружения

**Документация API:**
- **@nestjs/swagger** 7.1.17 — автоматическая генерация Swagger/OpenAPI документации

**Реактивное программирование:**
- **RxJS** 7.8.1 — библиотека для реактивного программирования

**Языки программирования:**
- **TypeScript** 5.3.3 — типизированный надмножество JavaScript

### Архитектура приложения

**Паттерны проектирования:**
- **MVC (Model-View-Controller)** — разделение логики на слои
- **Модульная архитектура** — организация кода в модули (NestJS)
- **Dependency Injection** — внедрение зависимостей
- **Repository Pattern** — работа с данными через репозитории (TypeORM)

**API стиль:**
- **RESTful API** — архитектурный стиль для веб-сервисов
- **JWT Authentication** — токен-базированная аутентификация
- **OAuth 2.0** — авторизация через Google

---

### Дополнительные инструменты

- **Swagger/OpenAPI** — интерактивная документация API
- **Environment Variables (.env)** — конфигурация через переменные окружения
- **CORS** — настройка кросс-доменных запросов
- **Validation Pipes** — валидация входящих данных
